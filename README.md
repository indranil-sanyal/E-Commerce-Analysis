# E-Commerce-Analysis
## Overview
This project analyzes shipping performance using historical order and shipment data from Nassau Candy Distributor. The objective is to identify delivery delays, regional bottlenecks, and logistics inefficiencies through EDA and interactive analytics.

## Key Focus Areas

Shipping lead time analysis

Delay frequency across routes and regions

Geographic bottleneck identification

Ship mode performance comparison

Outlier handling for extreme lead-time values

## Deliverables

Power BI Dashboard – interactive logistics insights

Research Paper – EDA, insights, and recommendations

Executive Summary – policy-oriented findings

## Tools Used

Power BI (DAX, data modeling)

CSV

Exploratory Data Analysis

## Outcome

The project demonstrates how data-driven analysis improves logistics visibility, reliability assessment, and decision-making for large distribution networks.
